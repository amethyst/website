pub struct TextureMetadata {
    pub sampler: SamplerInfo,
    pub mip_levels: u8,
    pub dynamic: bool,
    pub format: SurfaceType,
    pub size: Option<(u16, u16)>,
    pub channel: ChannelType,
}

Additional texture metadata that can be passed to the asset loader or added to the prefab.

Fields

sampler: SamplerInfo

The sampler info describes how to read from the texture, thus specifies filter and wrap mode. The default is nearest filtering (FilterMethod::Scale) and clamping (WrapMode::Clamp).

mip_levels: u8

Mipmapping levels. The default is one level.

dynamic: bool

Dynamic texture

format: SurfaceType

The surface type of the texture which describes the number of color channels and their size. In simpler words, this defines the color format, e.g. RGBA 32-bit.

The default is R8_G8_B8_A8.

size: Option<(u16, u16)>

The dimensions of the texture. Only needed for raw image data (TextureData::U8 etc).

channel: ChannelType

The channel type which describes the data format of the channels (e.g. how the red value is stored).

This is usually Srgb for color textures, normalmaps & similar mostly use Unorm (which represents a value between 0.0 and 1.0).
