Amethyst Branding Guidelines

This is how you use our brand

What is This Page?

This page is useful for anyone looking to use Amethyst assets for purposes such as featuring our logo in a news publication or for someone who simply wants to design something that needs to follow Amethyst's branding guidelines.

The Name

Amethyst should always be typed out with a capital A and lowercase letters for the rest. Examples of how not to style the name are amethyst, AMETHYST and AmEtHySt.

Logo

The logo is, contrary to other assets of the Amethyst Foundation, licensed under the CC-BY-ND 4.0 license and as such you're free to use the logo for any purpose with the limitation that you aren't allowed to modify it. The versions you are allowed to use without explicit permission from the Amethyst Foundation are those listed on this page.

Light Backgrounds

On light backgrounds, the standard logo with the Amethyst gradient should be used with transparency in the middle section. Download the standard logo

Dark Backgrounds

On dark backgrounds, the monochrome white verison of the logo may be used. Download the white logo

Any Background

Alternatively, you can also use the standard gradient logo with the middle section filled in with white on any background. Download the filled logo

Typography

Headers use the Arvo font while body content uses the Karla font. Primary text uses a color value of rgba(40, 38, 49, 0.85) while secondary text uses rgba(40, 38, 49, 0.65). All sizes are based off a 16px baseline.

Header Level 1 2.4rem Header Level 2 2.0rem Header Level 3 1.5rem Header Level 4 1.3rem Header Level 5 1.2rem Header Level 6 1.1rem Subtitle Text 1.4rem

The body text uses the primary text color and has a size of 1.1rem which makes it easily readable to anyone.

There is also a smaller version of the body text which can be used for things that don't require as much attention. It uses the secondary text color and has a size of 0.9rem.
